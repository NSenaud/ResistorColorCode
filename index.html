ResistorColorCode

This little OS X utility returns a resistor value from its color code. It is a very simple way to control a resistor when your working on electronic projects.

Current Features

Simply click on colors buttons to get a resistor value from its three first bands.

What's next?

I hope to be able to improve design, and I will add keyboard shortcuts. Next, why not to add Dictation support? If you have any idea, send to nicolas at senaud dot fr.
